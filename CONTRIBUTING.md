# Руководство по Вкладу

## Процесс разработки

1. Форкните репозиторий
2. Создайте ветку для функции (`git checkout -b feature/AmazingFeature`)
3. Зафиксируйте изменения (`git commit -m 'Add AmazingFeature'`)
4. Запушьте ветку (`git push origin feature/AmazingFeature`)
5. Откройте Pull Request

## Стандарты кода

- Используйте современный JavaScript/TypeScript
- Соблюдайте принципы Web6 архитектуры
- Тестируйте все частотные протоколы
- Документируйте новые возможности
